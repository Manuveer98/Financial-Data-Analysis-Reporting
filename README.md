# 📊 Financial Data Analysis & Reporting

This Power BI dashboard models a complete financial reporting solution using raw accounting data. It includes a custom-built **Income Statement**, KPI calculations, and country-wise performance comparisons across the years 2018–2020.

---

## 🔍 Project Overview

- **Tools Used**: Power BI Desktop , MS Excel
- **Data Sources**: Chart of Accounts, General Ledger, Inventory, Calendar  
- **Focus Areas**: Financial Statement Modeling, KPI Calculation, Country-Level Analysis
